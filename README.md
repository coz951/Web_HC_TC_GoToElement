# Web_HC_TC_GoToElement
網頁：赫綵臺中 前往指定元素

展示網頁:https://coz951.github.io/Web_HC_TC_GoToElement/

# 使用方式

1. CDN

```
<script src="https://coz951.github.io/Web_HC_TC_GoToElement/goToElement.js"></script>
```

2. 添加屬性

```
 data-gte-target="目標 ID" 
 data-gte-offset="位移 上方空間"
```

3. 目標添加 ID

```
<div id="目標 ID"></div>
```
